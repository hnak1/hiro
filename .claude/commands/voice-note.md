# /voice-note

## 呼び出し方
`/voice-note "テキスト"`

## 概要
音声メモや素早いメモをMarkdown形式に変換し、タグとアクションアイテムを付与して保存する。

## 処理手順

### 1. 日時付与
- 現在の日時を `YYYY-MM-DD HH:mm` 形式で付与

### 2. Markdown変換
- 話し言葉・断片的な文章を整形
- 箇条書き・見出しを適切に付与

### 3. キーワード抽出・タグ付与
- 主要キーワードを5個以内で抽出
- タグ形式（#タグ名）で付与

### 4. アクションアイテム抽出
- 「〜する」「〜確認」「〜調べる」などを検出
- チェックボックスリストで整理

### 5. 保存

出力フォーマット：
```markdown
---
date: YYYY-MM-DD HH:mm
tags: [#タグ1, #タグ2, ...]
---

# [自動生成タイトル]

[整形されたテキスト]

## アクションアイテム
- [ ] [アクション1]
- [ ] [アクション2]
```

`00_Inbox/voice-[YYYYMMDD-HHmm].md` として保存し、git commitする。
