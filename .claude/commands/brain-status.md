# /brain-status

## 呼び出し方
`/brain-status`

## 概要
brainの現在の健全性を一画面で確認する。

## 出力項目

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🧠 BRAIN STATUS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📁 総ノート数: N件
   00_Inbox:           N件（⚠️ N件滞留）
   10_Journal:         N件
   20_Projects:        N件
   30_Tech_Notes:      N件
   50_Business_Context: N件
   90_System:          N件
   99_Archives:        N件

📊 プロジェクト状況
   アクティブ: N件
   放置（7日以上）: N件

📅 活動
   直近30日の更新: N件
   最終commit: [日時]

💾 brainサイズ: N MB

🎯 推奨アクション
   1. [アクション]
   2. [アクション]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 処理手順
1. 各フォルダのファイル数をカウント
2. `git log` で直近30日の活動を集計
3. 7日以上更新のないプロジェクトを検出
4. `du -sh ~/brain` でサイズ取得
5. 状況に応じた推奨アクションを生成
