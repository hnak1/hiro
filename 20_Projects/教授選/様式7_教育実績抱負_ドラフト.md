### 3-1. 琉球大学着任後の卒前教育構想

**現状認識（琉球大学の文脈）:**
```
琉球大学医学部は沖縄・離島医療を担う医師の主要な育成機関である。
口腔外科領域においては[現状の課題を1〜2文で]。
```

**具体的な構想:**
```
① 口腔外科臨床実習の強化
   - 口腔がん早期発見・診断の実践的訓練を取り入れる
   - 全身管理（全身麻酔下手術）の見学機会を増やす

② チーム医療教育
   - 歯学・医学・看護の連携を体験するSP（模擬患者）演習を導入
   - 離島医療シミュレーション（へき地・遠隔診療を想定した演習）

③ グローバル教育
   - 英語での症例プレゼンテーション演習を必修化
   - アジア・太平洋地域の口腔外科医との交流プログラム検討
```

### 3-2. 大学院・研究指導構想

```
■ 目標: 5年以内に修了者 [　] 名・学術論文 [　] 報以上

■ 指導方針
① 琉球大学の強みである「島嶼医療・熱帯・感染症」との接点を持つ
  口腔外科研究テーマを開拓する
  （例: 口腔がんと環境・食習慣との関連、HIVと口腔疾患）
② 国際共同研究の機会を積極的に大学院生に提供する

■ メンタリング体制
- 月1回の個別面談（研究進捗・キャリア相談）
- 学会発表を年1回以上義務化
- 修了後のキャリア支援（就職・ポスドクのネットワーク活用）
```

### 3-3. 沖縄・離島医療人育成

```
沖縄県には[離島数]の島があり、口腔外科的疾患（悪性腫瘍・外傷・感染症）
への対応が医療均霑化の観点で重要な課題である。

■ 具体的取り組み構想
① 離島勤務のモチベーションを高める卒前教育プログラム
   - 離島医療の実情を知る「フィールド実習」（伊平屋島・与那国島等）
② 専攻医・若手口腔外科医の育成ルート整備
   - 琉球大学を中核とした県内研修病院ネットワーク
③ 遠隔指導・ICT活用による離島医師支援
   - 術前カンファレンスのオンライン化・遠隔病理診断連携
```
