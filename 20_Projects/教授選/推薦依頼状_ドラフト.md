## 推薦者選定チェックリスト

推薦者に求められる条件（選考委員の評価目線）:

| 条件 | 優先度 | 備考 |
|------|--------|------|
| 応募者の研究・臨床能力を直接評価できる | 必須 | 指導教員・共同研究者が理想 |
| 口腔外科または関連領域の権威 | 高 | 学会役員・教授・学長クラス |
| **琉球大学・沖縄との接点がある** | **戦略的優先** | 地域連携の文脈で選考委員に響く（#17参照） |
| 国際的知名度がある | 加点 | 海外の共著者・招待先 |
| 応募者と利害関係なし（採用後に上司になるポジション等は避ける） | 必須 | 不適切な利益供与と見なされるリスク |

### 📍 戦略ノート（2026-03-15 追記 by #17）

> **「推薦者は地域連携を考えて沖縄の先生がいいだろう」**

沖縄在住の先生が推薦者になることの戦略的価値:

1. **信頼性の補強**: 琉球大学選考委員と顔見知りの可能性が高く、推薦の重みが増す
2. **地域連携の実証**: 「沖縄の医療界に既にネットワークがある」というシグナルになる
3. **着任意欲の裏付け**: 地元の先生が推薦 = 本気で沖縄に根付く意思の証拠
4. **様式6/7/8との一貫性**: 3書類で「沖縄・離島医療への貢献」を謳っており、推薦者の地域性がストーリーを強化

**推薦者候補の優先順位（改訂）**:

```
優先度A+（最高）: 沖縄在住の教授・権威 → 研究+地域両面で証言できる ★新設★
優先度A（高）: 指導教授・元ボス → 研究・人物両面を証言できる
優先度B（高）: 共同研究者（上位）+ 沖縄接点あり → 最強の組み合わせ
優先度C（中）: 学会役員（沖縄外） → 客観的第三者として信頼性高い
優先度D（低）: 同僚・同期 → 客観性の担保が難しい
```

**沖縄の先生候補を探すヒント**:
- 日本口腔外科学会 九州・沖縄支部の役員・顧問
- 琉球大学と共著論文がある先生
- 沖縄県立病院・那覇市立病院の口腔外科部長クラス
- 過去の学会・研究会で面識のある沖縄の先生

## 推薦書の内容についての依頼ポイント（先生への参考情報として送付）

推薦書に含めていただくと評価が高まる要素:

1. **応募者との関係と評価歴**（いつから・どの立場で指導/共同研究したか）
2. **研究能力の具体的評価**（IF合計・独自性・将来性への言及）
3. **教育・指導力**（大学院生・後輩への指導スタイル）
4. **人柄・リーダーシップ**（選考委員は人物評価も重視）
5. **琉球大学・地域医療との適合性**（口腔がん・離島医療への貢献可能性）

> ※ 上記を「先生への参考メモ」として一緒に送付すると、
> 推薦書の質が上がり、且つ先生の執筆負担も軽減される。
